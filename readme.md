# API NODE.JS USANDO MONGODB, EXPRESS E JWT

## Requisitos

* Node.JS: v14.20.1 ou superior
* Npm: 6.14.17 ou superior

## Instalar as dependencias

    npm install

## Iniciar a aplicação

    npm run start
