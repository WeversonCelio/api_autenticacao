# API DE AUTENTICAO NO SISTEMA

Neste projeto foi usado Mongodb, express, node.js e jwt.

## Requisitos

* Node.JS: v14.20.1 ou superior
* Npm: 6.14.17 ou superior

## Instalar as dependencias

    npm install

## Iniciar a aplicação

    npm run start
